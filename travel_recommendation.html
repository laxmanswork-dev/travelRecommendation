About Us

Travel Explorer is a simple travel recommendation platform created as a final project for the Coursera JavaScript Programming Essentials course. It demonstrates how to use JavaScript, JSON data, and dynamic DOM manipulation to build interactive web pages.

Use the search bar above to explore different destinations by keyword, and instantly see recommended places with images and short descriptions.
